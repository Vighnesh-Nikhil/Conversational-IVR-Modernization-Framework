# Conversational IVR Modernization Framework

## Project Overview
This project focuses on modernizing a traditional IVR (Interactive Voice Response) system by integrating conversational AI capabilities. Users can interact with the system using natural language instead of keypad-based inputs.

## Features
- Menu-driven IVR system (Milestone 2)
- Conversational AI interface (Milestone 3)
- Session management using session_id
- Hospital lab report IVR use case
- Natural language intent detection
- Backend developed using FastAPI

## Testing (Milestone 4)
- Unit testing using Pytest
- API testing using FastAPI TestClient
- IVR flow validation
- Basic performance testing

## Deployment
